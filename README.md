# Modern Minimal Portfolio

Template de portfólio em React + Vite + Tailwind.

## Como rodar localmente
```bash
npm install
npm run dev
```

## Deploy no GitHub Pages
1. Edite `vite.config.js` e troque `base: '/seurepo/'` pelo nome do seu repositório.
2. Commit e push para `main`.
3. O GitHub Actions publicará automaticamente em **Settings > Pages**.

## Deploy no Vercel
1. Importar repositório no Vercel.
2. Build Command: `npm run build`.
3. Output Directory: `dist`.
